Discord Username Checker (Pomelo)

How to use:
1. Paste one username per line in the "Usernames" box (bulk supported).
2. (Optional) Add proxy list (one proxy per line, format: http://ip:port or http://user:pass@ip:port).
3. Click "Start Bulk Check".

Important notes:
• This tool uses a public third-party endpoint for realistic checking[](https://api.lixqa.de/v3/discord/pomelo/{username}).
• Results: 2 = Available, 3/4 = Taken/Reserved, other = Error.
• Proxies help avoid rate limits and improve speed/privacy.
• Be respectful — avoid excessive requests. Discord may rate-limit or flag heavy usage.
• Usernames must be 2-32 characters (a-z, 0-9, _, .).
• This is a client-side HTML file — save it as discord-username-checker.html and open in your browser.
